**XLeRobot**은 **XL + LeRobot**의 합성어로, SO-ARM을 기반으로 한 듀얼암 모바일 가정용 로봇입니다.

<Callout type="tip" title="🚀 Embodied AI를 모든 가정에!">
  iPhone보다 저렴한 가격으로 가정용 로봇을 경험할 수 있습니다. 조립 시간은 4시간 미만!
</Callout>

## 🎯 XLeRobot이란?

XLeRobot은 [LeRobot](https://github.com/huggingface/lerobot), [SO-100/SO-101](https://github.com/TheRobotStudio/SO-ARM100), [LeKiwi](https://github.com/SIGRobotics-UIUC/LeKiwi), [Bambot](https://github.com/timqian/bambot)의 거인들 위에 구축된 프로젝트입니다.

### "LeRobot" 코어의 장점

- **저렴한 재료** 💴: 90% 3D 프린팅 부품과 저렴한 모터 및 전자부품
- **쉬운 조립** 🔨: 2개의 SO-101 암 조립 및 모터 설정에 67분
- **플러그 앤 플레이** 🧩: 간단한 pip install과 몇 줄의 코드로 로봇 실행
- **활발한 커뮤니티** 🌍: 세계 최대 저가형 로봇공학 커뮤니티
  - 여러 SOTA VLA 모델, 데이터셋, 도구 제공
  - 10,000+ 멤버의 브레인스토밍 및 토론

### "XL" 향상의 이유

- 🏠 저렴하고 안정적이며 조립이 쉬운 듀얼암 가정용 로봇의 부재
- 🖨️ 전통적인 3D 프린팅 샤시의 내구성, 안정성, 적재 용량 제한
- ⚡ DIY 모바일 로봇의 전원 공급 문제

## 주요 특징

### 🤖 하드웨어

- **듀얼암 시스템**: 2개의 SO-101 로봇 팔 (각 6축)
- **모바일 플랫폼**: 전방향 이동이 가능한 홀로노믹 드라이브
- **IKEA 카트 베이스**: 안정적이고 튼튼한 구조
- **멀티 카메라**: RGB 카메라 또는 RealSense RGBD 지원
- **컴팩트 디자인**: 가정용으로 최적화된 크기

### 🧠 소프트웨어

- **LeRobot 완벽 지원**: 모든 LeRobot 기능 활용 가능
- **다양한 제어 방식**: 키보드, Xbox 컨트롤러, Switch 조이콘, VR(Quest3)
- **실시간 제어**: 블루투스 연결, WiFi 불필요, 0 지연시간
- **시뮬레이션 지원**: ManiSkill 시뮬레이션 환경 제공
- **다양한 센서**: 깊이, 세그멘테이션, 촉각 센서 지원

## 🌟 전체 장점

### 비용 효율성
- 완전한 빌드 비용 $660
- SO-101 Arm과 LeKiwi에서 $250로 업그레이드 가능

### 쉬운 업그레이드
- **LeKiwi**와 **SO-100/SO-101**에서 간편한 업그레이드
- 모터 ID 변경이나 수정 불필요
- 동일한 테이블탑 단일/듀얼암 설정
- SO-101 암의 정책을 직접 전송 가능

### 실용성 및 신뢰성
- $30,000 클로즈드 소스 로봇과 비교 가능한 많은 일상 작업 수행
- [HuggingFace LeRobot 월드와이드 해커톤](https://huggingface.co/spaces/LeRobot-worldwide-hackathon/all-winners)에서 더 많은 작업 시연

### 풍부한 오픈소스 리소스
- LeRobot의 플러그 앤 플레이 코드
- 방대한 AI 모델 라이브러리
- 활발하고 성장하는 기여자 커뮤니티

### ⚠️ 안전성 ⚠️
- 저토크 모터, 짧은 암 길이, 휠 기반 설계로 인해 물리적으로 인간에게 해를 끼칠 가능성이 거의 없음
- 여전히 많은 가정용 작업 수행 능력 유지

## 제한사항

(이 가격에 무엇을 더 바랄 수 있을까요?)

- 🔒 **고정 높이**: 안정적인 리프팅 플랫폼을 추가하면 비용과 조립 난이도가 크게 증가
- 📏 **작은 작업 공간**: Aloha에 비해 작음 - SO-101 작업 공간을 최대화하려 하지만 암 크기 제한 (~40cm 도달 거리)
- ⚖️ **제한된 페이로드**: 단일 암당 600~1000g
- 🤹 **손재주 제한**: 현재 인핸드 손재주를 위해 설계되지 않음
- 🏋️ **무거운 작업 제한**: 암당 1kg 이상
- 🏃 **동적 동작 제한**: 공 잡기 등

그러나 모든 것을 고려했을 때 — 비용, 커뮤니티 지원, 조립 용이성, 실용성 — XLeRobot은 실내 응용을 위한 가장 매력적인 저가형 로봇 중 하나입니다!
